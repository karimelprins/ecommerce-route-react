# GitHub Upload Guide

## Recommended repository name

```txt
ecommerce-route-react
```

## Description

```txt
Professional React e-commerce app using Route E-commerce API with products, auth, cart, wishlist, and checkout flow.
```

## Upload steps

```bash
git init
git add .
git commit -m "Initial commit: professional React e-commerce app"
git branch -M main
git remote add origin https://github.com/karimelprins/ecommerce-route-react.git
git push -u origin main
```

## After upload

1. Add screenshots to `screenshots/`
2. Deploy on Vercel
3. Add the live demo link to README
4. Pin the repo on GitHub profile
5. Add it to LinkedIn Featured
